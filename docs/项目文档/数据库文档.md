# QFriendly 数据库文档

## 概览

- 数据库: PostgreSQL `qfriendly`
- ORM: Drizzle ORM (`drizzle-orm` + `drizzle-kit`)
- Schema 定义: `lib/db/schema.ts`
- 总表数: 17 张（认证 4 张 + 业务 5 张 + 统计 4 张 + 配置 4 张）

## 表分组

| 分组 | 表名 | 用途 |
|------|------|------|
| 认证 | `user`, `session`, `account`, `verification` | Better Auth 托管，自动生成 |
| 业务 | `subscriptions`, `profiles`, `conversations`, `messages` | 订阅、用户档案、对话 |
| 统计 | `analytics_events`, `analytics_daily_stats`, `analytics_monthly_stats`, `analytics_retention` | 埋点事件 + 聚合 |
| 配置 | `expert_prompts` | 专家提示词（System Prompt/欢迎语/切换模板） |

## 认证表

### user
Better Auth 用户主表。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | text | PK | 用户 ID（Better Auth 生成） |
| name | text | NOT NULL | 用户名 |
| email | text | NOT NULL, UNIQUE | 邮箱 |
| email_verified | boolean | NOT NULL, DEFAULT false | 邮箱是否已验证 |
| image | text | | 头像 URL |
| created_at | timestamp | NOT NULL | 注册时间 |
| updated_at | timestamp | NOT NULL | 更新时间 |

### session
Better Auth 会话表。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | text | PK | 会话 ID |
| expires_at | timestamp | NOT NULL | 过期时间 |
| token | text | NOT NULL, UNIQUE | 会话令牌 |
| ip_address | text | | 客户端 IP |
| user_agent | text | | 客户端 UA |
| user_id | text | NOT NULL, FK→user.id CASCADE | 所属用户 |

### account
Better Auth 账户表（provider 凭证存储）。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | text | PK | 账户 ID |
| account_id | text | NOT NULL | Provider 侧账户 ID |
| provider_id | text | NOT NULL | Provider 类型（如 credential） |
| user_id | text | NOT NULL, FK→user.id CASCADE | 所属用户 |
| access_token | text | | OAuth token |
| refresh_token | text | | OAuth refresh token |
| password | text | | 密码哈希（email/password 登录） |
| created_at | timestamp | NOT NULL | |
| updated_at | timestamp | NOT NULL | |

### verification
Better Auth 验证表（邮箱验证 token 等）。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | text | PK | |
| identifier | text | NOT NULL | 验证目标（如邮箱） |
| value | text | NOT NULL | 验证码/令牌 |
| expires_at | timestamp | NOT NULL | 过期时间 |

## 业务表

### subscriptions
用户订阅记录。通过 PayPal 支付。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | uuid | PK, DEFAULT gen_random | |
| user_id | text | NOT NULL, FK→user.id CASCADE | |
| paypal_subscription_id | text | NOT NULL, UNIQUE | PayPal 订阅 ID |
| paypal_plan_id | text | NOT NULL | PayPal Plan ID |
| variant_name | text | NOT NULL | 等级: start/pro/ultra/admin |
| status | subscription_status | NOT NULL, DEFAULT active | active/cancelled/expired/suspended |
| current_period_start | timestamp | | 当前账期开始 |
| current_period_end | timestamp | | 当前账期结束 |
| created_at | timestamp | DEFAULT now | |
| updated_at | timestamp | DEFAULT now | |

管理员通过 `variant_name = 'admin' AND status = 'active'` 识别。

### profiles
用户扩展档案（1:1 user，懒创建）。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | uuid | PK, DEFAULT gen_random | |
| user_id | text | NOT NULL, UNIQUE, FK→user.id CASCADE | |
| nickname | text | | 昵称 |
| trial_used | integer | DEFAULT 0 | 已用免费消息数 |
| daily_limit | integer | | 自定义日限额（覆盖方案默认值） |
| created_at | timestamp | DEFAULT now | |

### conversations
对话会话记录。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | uuid | PK, DEFAULT gen_random | |
| user_id | text | NOT NULL, FK→user.id CASCADE | |
| expert | expert | NOT NULL | evan/liam/noah/adrian |
| language | language | NOT NULL, DEFAULT en | en/zh |
| title | text | DEFAULT 'New Conversation' | 对话标题 |
| created_at | timestamp | DEFAULT now | |
| updated_at | timestamp | DEFAULT now | |

索引: `(user_id)` 隐式支持按用户查询。

### messages
对话消息记录。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | uuid | PK, DEFAULT gen_random | |
| conversation_id | uuid | NOT NULL, FK→conversations.id CASCADE | |
| role | message_role | NOT NULL | user/assistant |
| content | text | NOT NULL | 消息正文 |
| tokens | integer | | Token 用量（当前未使用） |
| created_at | timestamp | DEFAULT now | |

索引: `(conversation_id)`, `(conversation_id, created_at)`

## 统计表

### 数据流向

```
analytics_events (原始事件)
    ↓ 定时聚合
analytics_daily_stats (日级)
    ↓ 定时聚合
analytics_monthly_stats (月级)

analytics_retention (留存快照，独立计算)
```

### analytics_events
原始事件流水表。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | uuid | PK | |
| event_type | text | NOT NULL | 事件类型: payment_completed/page_view 等 |
| user_id | text | FK→user.id SET NULL | 触发用户（可空） |
| payload | jsonb | DEFAULT {} | 事件负载数据 |
| created_at | timestamp | DEFAULT now | |

索引: `(event_type, created_at)`

### analytics_daily_stats
日级聚合统计。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | uuid | PK | |
| date | date | NOT NULL | 统计日期 |
| metric_key | text | NOT NULL | 指标名: dau/message_count/pv/uv/payment_total 等 |
| metric_value | numeric | NOT NULL, DEFAULT 0 | 指标值 |

唯一索引: `(date, metric_key)`

### analytics_monthly_stats
月级聚合统计。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | uuid | PK | |
| year_month | text | NOT NULL | 格式: YYYY-MM |
| metric_key | text | NOT NULL | 指标名 |
| metric_value | numeric | NOT NULL, DEFAULT 0 | 指标值 |

唯一索引: `(year_month, metric_key)`

### analytics_retention
用户留存率快照。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | uuid | PK | |
| cohort_date | date | NOT NULL | 队列日期 |
| day_n | smallint | NOT NULL | 第 N 天 (1/7/30) |
| retention_rate | numeric | NOT NULL, DEFAULT 0 | 留存率 |

唯一索引: `(cohort_date, day_n)`

## 配置表

### expert_prompts
AI 专家提示词配置表，支持管理员在线编辑 System Prompt、欢迎语和切换过渡模板，中英双语各一份。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | uuid | PK, DEFAULT gen_random | |
| expert | expert | NOT NULL | 专家标识: evan/liam/noah/adrian |
| language | language | NOT NULL | 语言: en/zh |
| prompt_type | text | NOT NULL | 类型: system/welcome/switch |
| content | text | NOT NULL | 提示词正文 |
| updated_at | timestamp | DEFAULT now | 更新时间 |

唯一约束: `(expert, language, prompt_type)`，最多 24 行（4 专家 × 2 语言 × 3 类型）。
读取时若表中无对应记录，回退到代码中的硬编码默认值（降级可用）。

## 枚举类型

| 枚举名 | 值 | 用途 |
|--------|-----|------|
| expert | evan, liam, noah, adrian | 专家标识 |
| language | en, zh | 对话语言 |
| message_role | user, assistant | 消息角色 |
| subscription_status | active, cancelled, expired, suspended | 订阅状态 |
